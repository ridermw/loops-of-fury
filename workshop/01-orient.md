# 01 - Orient Copilot CLI

## Goal

Use Copilot CLI to understand the codebase and the failing checks before asking for a fix.

## Done-condition

You have a concise explanation of:

- What the app is supposed to do.
- Which tests fail.
- Which implementation areas are likely responsible.

## Suggested prompt

Paste `prompts/copilot-cli/01-orient.md` into Copilot CLI.

## Discussion

Ask participants:

- Did the agent inspect the tests before proposing changes?
- Did it identify a machine-checkable done-condition?
- Did it avoid jumping straight to edits?
