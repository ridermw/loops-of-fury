The lab app

Domain

Tiny inventory reservation service.

Verifier

Node's built-in test runner.

Goal

Make npm test pass.

Intentional bugs

  • SKU normalization misses spaces and underscores.
  • Blank SKUs are accepted.
  • Partial reservations lose reserved quantity.
  • Retries double-decrement stock.
  • Backorder summaries report requested quantity, not shortage.

Lab 1: orient

Use prompts/copilot-cli/01-orient.md.

No edits yet. Ask Copilot CLI to explain the app, failures, done-condition, and smallest safe scope.

Lab 2: maker

Use prompts/copilot-cli/02-maker-fix-tests.md.

The maker can edit only src/inventory.js, must not weaken tests, and stops when npm test passes.

Lab 3: checker

Use prompts/copilot-cli/03-checker-review.md.

The checker returns PASS or FAIL with evidence. The maker does not mark itself done.

Lab 4: loop contract

Use prompts/copilot-cli/04-loop-contract.md and workshop/loop-contract-template.md.
