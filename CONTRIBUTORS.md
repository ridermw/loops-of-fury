# Contributors

## Upstream maintainer

- `ridermw`

`ridermw/loops-of-fury` is the instructor-owned upstream workshop repository. The upstream maintainer may push directly to `main`.

## Workshop participants

Workshop participants should fork this repository and do all lab work in their own fork:

```powershell
gh repo fork ridermw/loops-of-fury --clone=true
cd loops-of-fury
```

Participants are not expected to open pull requests back to the upstream repo. Upstream PRs may be closed automatically so the workshop source stays clean.

## Contribution policy

This repository is primarily a workshop artifact, not an open contribution project. If contribution rules change later, this file should be updated first.
